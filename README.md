# Termux Keyboard App

A custom Android input method keyboard for Termux with a Mr. Robot inspired hacker terminal visual theme.

## Features

- Termux-focused shortcut keyboard
- ESC, TAB, CTRL, ALT keys
- Slash, pipe, tilde, dollar shortcuts
- Direction keys
- ENTER key
- CTRL+C, CTRL+D, CTRL+L shortcuts
- Quick command buttons:
  - `ls`
  - `cd`
  - `clear`
  - `pkg update && pkg upgrade`
- Dark hacker keyboard theme
- Neon green terminal-style text
- Monospace visual style

## Build with GitHub Actions

Push this project to a GitHub repository, then run:

`.github/workflows/02-build-termux-keyboard-apk.yml`

The APK will be uploaded as an artifact named:

`termux-keyboard-debug-apk`
